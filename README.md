# Admin Dashboard UI with Login and User Management

## Project Overview
This is a simple Admin Dashboard UI project made using HTML and CSS.
The main goal of this project is to learn basic dashboard design and understand how an admin panel looks and works. 
This project is only front-end based.
It project on layout, design, and structure of the dashboard.

## Features
- Login page with email/phone and password
- Admin dashboard layout
- User Management table
- shows user information:
 - Name 
 - Email
 - Role(Admin or User)
 - Status(Active or Inactive)
 - Simple and clean user interface 

 ## Technologies Used 
 - HTML5 
 - CSS3

 ## Project Sections 
 - Login Form 
 - Dashboard Page
 - User Management Section

 ## ## How to Setup the Project Locally
1. Download or clone the project from the repository.
2. Extract the project folder (if downloaded as ZIP).
3. Open the project folder in any code editor (VS Code recommended).
4. Open the `login.html` file in a web browser.
5. The project will run locally in the browser.

No additional setup or installation is required.

## Version Details
- HTML: HTML5
- CSS: CSS3
- Browser: Google Chrome / Microsoft Edge
- Editor: Visual Studio Code
